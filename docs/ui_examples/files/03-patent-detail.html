Abstract Copy
A neural network system and method for real-time object detection in autonomous vehicle applications. The system comprises an adaptive feature extraction module that dynamically adjusts receptive field sizes based on input image characteristics, a multi-scale attention mechanism for improved small object detection, and a lightweight inference engine optimized for edge deployment. The architecture achieves sub-10ms latency while maintaining detection accuracy above 95% mAP on standard autonomous driving benchmarks. The system further includes a novel temporal consistency module that leverages sequential frame information to reduce false positives and improve tracking stability in dynamic environments.
Claims (24) Expand All
1 Independent

A computer-implemented method for real-time object detection comprising: receiving an input image from a sensor array; processing the input image through an adaptive feature extraction module configured to dynamically adjust receptive field parameters; applying a multi-scale attention mechanism to extracted features; generating object detection outputs including bounding boxes and classification scores; and outputting detection results in sub-10ms latency.

2 Dependent

The method of claim 1, wherein the adaptive feature extraction module comprises a plurality of convolutional layers with dynamically adjustable kernel sizes based on input image resolution.

3 Dependent

The method of claim 1, wherein the multi-scale attention mechanism comprises a self-attention layer operating across multiple spatial scales simultaneously.

4 Dependent

The method of claim 1, further comprising a temporal consistency module that processes sequential frames to reduce false positive detections.
